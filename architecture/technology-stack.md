# Technology Stack

#### Technology Stack

**Blockchain:**

* BNB Chain (BSC Mainnet - Chain ID: 56)
* Solidity ^0.8.20 for smart contracts
* OpenZeppelin Contracts v5.0 for security standards
* Hardhat development environment for compilation and deployment

**Frontend:**

* Next.js 14 with App Router
* React 18 with TypeScript
* ethers.js v6 for Web3 interactions
* TailwindCSS for styling
* FontAwesome for icons

**Infrastructure:**

* IPFS via Pinata for decentralized metadata storage
* MongoDB for optional off-chain caching and analytics
* BSC RPC endpoints for blockchain queries
