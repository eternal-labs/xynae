# Assets

This directory contains visual assets for the Xynae project.

## Files

Place your PNG files in `assets/images/`:

- **`logo.png`** - Main Xynae logo (used in README) - Recommended: 200x200px or larger
- **`social-preview.png`** - Social preview image for GitHub - Must be 1200x630px
- **`favicon.png`** - Favicon image - Recommended: 32x32px or 64x64px

## Setting Up Social Preview on GitHub

To set the social preview image (the image that appears when sharing your repository link):

1. Go to your repository on GitHub: https://github.com/eternal-labs/xynae/settings
2. Click **Settings** â†’ **General**
3. Scroll down to **Social preview**
4. Click **Upload an image**
5. Upload `assets/images/social-preview.png` (must be 1200x630px PNG)

## Using the Logo

The logo is embedded in the README using:
```markdown
![Xynae Logo](assets/images/logo.png)
```

## Favicon

Place your `favicon.png` in `assets/images/`. For web usage, you can convert it to `.ico` format using [favicon.io](https://favicon.io/).
